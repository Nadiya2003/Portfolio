# Next.js Portfolio

A stunning personal portfolio built with **Next.js 14**, **TypeScript**, **Tailwind CSS**, and **Framer Motion**.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Tech Stack

- **Next.js 14** (App Router)
- **TypeScript**
- **Tailwind CSS** 3.4
- **Framer Motion** for animations
- **Lucide React** for icons

## Features

- ✨ Intro sequence animation with skip
- 🎯 Smooth scroll navigation with active section tracking
- 🖼️ Portfolio gallery with category filtering & before/after slider
- 💫 Glassmorphism UI design system
- 🧲 Magnetic button effects
- 📊 Animated skill progress rings & bars
- 💬 Scrolling testimonials carousel
- 📬 Contact form with success state
- 🌙 Full dark mode design
- 📱 Fully responsive
